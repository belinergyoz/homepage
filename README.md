# Belinergy o.z. — GitHub Pages

Stránka je pripravená pre nasadenie na GitHub Pages. Súbory lokálne nájdete v koreňovom adresári repozitára.

Rýchly návod:

- Skontrolujte, že `index.html` je v koreňovom adresári repozitára.
- Pushnite zmeny do vetvy `main` (alebo `gh-pages`, ak to preferujete).
- V GitHub repozitári otvorte `Settings → Pages` a vyberte vetvu (napr. `main`) a adresár `/ (root)`.
- Ak chcete publikovať ako projektovú stránku (https://USERNAME.github.io/REPO/), nastavte v `<head>` `base href="/REPO/"` alebo použite relatívne odkazy.
- Pre vlastnú doménu pridajte súbor `CNAME` s vašou doménou.

Ak chcete, môžem:

- pridať `CNAME` s vašou doménou
- nastaviť `base href` pre projektovú stránku
- vytvoriť vetvu `gh-pages` a commitnut tam súbor automaticky
